CSS动画类型展示

缩放动画

使元素放大和缩小，创造出动态效果。

@keyframes scale { 0% { transform: scale(0.5); opacity: 0; } 50% { transform: scale(1.2); } 100% { transform: scale(1); opacity: 1; } } .scale-box { animation-name: scale; animation-duration: 1.5s; animation-iteration-count: infinite; animation-direction: alternate; }

旋转动画

使元素围绕中心点旋转。

@keyframes rotate { from { transform: rotate(0deg); } to { transform: rotate(360deg); } } .rotate-box { animation-name: rotate; animation-duration: 2s; animation-iteration-count: infinite; animation-timing-function: linear; }

弹跳动画

模拟物体弹跳的物理效果。

@keyframes bounce { 0%, 20%, 50%, 80%, 100% { transform: translateY(0); } 40% { transform: translateY(-30px); } 60% { transform: translateY(-15px); } } .bounce-box { animation-name: bounce; animation-duration: 1.5s; animation-iteration-count: infinite; }

颜色变化动画

平滑过渡元素的背景颜色。

@keyframes colorChange { 0% { background-color: #9b59b6; } 50% { background-color: #f39c12; } 100% { background-color: #9b59b6; } } .color-box { animation-name: colorChange; animation-duration: 3s; animation-iteration-count: infinite; }

呼吸动画

模拟呼吸效果，缓慢缩放和改变透明度。

@keyframes breathe { 0%, 100% { transform: scale(1); opacity: 1; } 50% { transform: scale(1.2); opacity: 0.7; } } .breathe-box { border-radius: 50%; animation-name: breathe; animation-duration: 2s; animation-iteration-count: infinite; }

脉冲动画

创建向外扩散的脉冲效果，常用于通知或强调。

@keyframes pulse { 0% { box-shadow: 0 0 0 0 rgba(255, 105, 180, 0.7); } 70% { box-shadow: 0 0 0 15px rgba(255, 105, 180, 0); } 100% { box-shadow: 0 0 0 0 rgba(255, 105, 180, 0); } } .pulse-box { border-radius: 50%; animation-name: pulse; animation-duration: 2s; animation-iteration-count: infinite; }
